# react-native-veel-webview

Veel webview

## Installation

```sh
npm install rn-veel-webview
```

## Usage

```js

import VeelWebView from 'rn-veel-webview';

return (
    <View style={styles.container}>
        <VeelWebView>
    </View>
)

```
